# ⚽ Bagatelle Sorteio

App de sorteio equilibrado para o Bagatelle Football Society.

---

## 🚀 Como colocar no ar (passo a passo)

### 1. Criar conta no GitHub
- Acesse: https://github.com
- Clique em **Sign up**
- Crie sua conta com email e senha

### 2. Criar repositório no GitHub
- Após logar, clique no **+** no canto superior direito
- Clique em **New repository**
- Nome: `bagatelle-sorteio`
- Deixe como **Public**
- Clique em **Create repository**

### 3. Subir os arquivos
- Na página do repositório, clique em **uploading an existing file**
- Selecione TODOS os arquivos desta pasta (incluindo subpastas)
- Clique em **Commit changes**

> ⚠️ Suba respeitando a estrutura de pastas:
> - `src/App.jsx` e `src/main.jsx`
> - `public/manifest.json`, `public/icon-192.png`, `public/icon-512.png`
> - `index.html`, `package.json`, `vite.config.js`

### 4. Criar conta no Vercel
- Acesse: https://vercel.com
- Clique em **Sign up**
- Escolha **Continue with GitHub**
- Autorize o acesso

### 5. Fazer o deploy
- No Vercel, clique em **Add New Project**
- Selecione o repositório `bagatelle-sorteio`
- Mantenha as configurações padrão (ele detecta React/Vite automaticamente)
- Clique em **Deploy**
- Aguarde ~2 minutos

### 6. Seu link estará pronto!
- O Vercel vai gerar um link tipo: `bagatelle-sorteio.vercel.app`
- Compartilhe no grupo do WhatsApp!

---

## 📱 Instalar como app no iPhone

1. Abra o link no **Safari** (não Chrome)
2. Toque no botão de **compartilhar** (quadrado com seta pra cima)
3. Role e toque em **"Adicionar à Tela de Início"**
4. Toque em **Adicionar**
5. O ícone do Bagatelle aparece na tela! ⚽

---

## 📱 Instalar como app no Android

1. Abra o link no **Chrome**
2. Toque nos **3 pontos** no canto superior direito
3. Toque em **"Adicionar à tela inicial"**
4. Toque em **Adicionar**
